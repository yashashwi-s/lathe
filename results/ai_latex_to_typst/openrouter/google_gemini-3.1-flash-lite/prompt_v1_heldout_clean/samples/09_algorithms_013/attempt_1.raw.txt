= Algorithm
This sample contains algorithmic pseudocode extracted from a source-backed LaTeX benchmark dataset. It is wrapped in a minimal article document for pdfLaTeX validation.

#figure(
  caption: [Source-backed algorithmic procedure],
  block[
    #set enum(numbering: "1.")
    + Initialize actor network $mu_theta$ and critic networks $Q_(omega_1)$ and $Q_(omega_2)$.
    + Initialize corresponding target networks: $theta' arrow.l theta$, $omega'_1 arrow.l omega_1$, $omega'_2 arrow.l omega_2$ and choose $rho_"p" in (0, 1)$.
    + Initialize replay buffer $cal(R)$.
    + Choose a batch size $K$, a gradient based optimization algorithm and a corresponding learning rate $lambda_"actor", lambda_"critic" > 0$ for both optimization procedures, a time step size $Delta t$ and a stopping criterion.
    + Choose standard deviation exploration noise $sigma_"expl"$ and lower and upper action bounds $a_"low", a_"high"$.
    + *Repeat*
      + Select clipped action and step the environment dynamics forward.
        $a = "clip"(mu_theta(s) + epsilon, a_"low", a_"high"), quad epsilon tilde cal(N)(0, sigma_"expl")$
      + Observe next state $s'$, reward $r$, and done signal $d$ and store the tuple $(s, a, r, s', d)$ in the replay buffer.
      + #if true [
        *If* $s'$ is terminal
        + Reset trajectory.
        *End If*
      ]
      + *For* $j$ in range("update frequency")
        + Sample batch $cal(B) = {(s^((k)), a^((k)), r^((k)), s'^((k)), d^((k)))}_k=1^K$ from replay buffer.
        + Compute targets (Clipped Double Q-learning and policy smoothing).
          $y(r, s', d) = r + (1-d) min_(i=1, 2) { Q_(omega'_i)(s', tilde(a)) }, quad tilde(a) = "clip"(mu_(theta')(s) + epsilon, a_"low", a_"high"), quad epsilon tilde cal(N)(0, sigma_"target")$
        + Estimate critic gradient $nabla_omega L(Q_omega^(mu_theta))$ by
          $nabla_(omega_i) (1/K sum_(k=1)^K (Q_(omega_i)(s^((k)), a^((k))) - y(r^((k)), s'^((k)), d^((k))) )^2), quad "for" i=1,2$
        + Update the critic parameters $omega_i$ based on the optimization algorithm.
        + #if true [
          *If* $j "mod" "policy delay frequency" = 0$
          + Estimate actor gradient $nabla_theta J(mu_theta)$ by
            $nabla_theta (1/K sum_(k=1)^K Q_(omega_1)(s^((k)), mu_theta(s^((k))))) $
          + Update the actor parameters $theta$ based on the optimization algorithm.
          + Update target networks softly:
            $theta' arrow.l rho_"p" theta' + (1 - rho_"p")theta, quad omega'_i arrow.l rho_"p" omega'_i + (1 - rho_"p")omega_i, quad "for" i=1, 2$
          *End If*
        ]
      + *End For*
    + *Until* stopping criterion is fulfilled.
  ]
)
